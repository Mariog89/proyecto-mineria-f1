## 💻 EJEMPLO RÁPIDO

```python
import sys
sys.path.insert(0, 'scripts')
from genetic_optimization import create_ga_optimizer_for_model

optimizer = create_ga_optimizer_for_model(
    model_pipeline,
    X_train, y_train,
    param_space={
        'clf__C': ('float', 0.01, 100, 0.1),
        'clf__max_depth': ('int', 3, 20, 1),
    },
    generations=15,
    population_size=20
)

result = optimizer.optimize()
print(f"Best F1: {result['best_fitness']:.4f}")
```

## 📊 CARACTERÍSTICAS GA

| Feature | Descripción |
|---------|-------------|
| **Mutación** | Cambio aleatorio de parámetros |
| **Crossover** | Intercambio de genes |
| **Selección** | Por torneo (tournament selection) |
| **Elitismo** | Preserva mejores soluciones |
| **Tipos** | int, float, categorical |
| **Tracking** | Evolución por generación |
| **CV** | Evaluación con validación cruzada |

## 🔄 COMPARACIÓN MÉTODOS

```
GridSearch: Lento, determinístico, exhaustivo
Optuna:     Muy rápido, probabilístico, excelente
GA:         Medio, probabilístico, muy bueno

Usar GA cuando GridSearch sea muy lento
```
